# millerroofingnj
Miller Roofing Estimation and Proposal Generation System

This project is a Python-based application that generates roofing estimates using a Good, Better, Best pricing model. It calculates costs, applies markup, and produces a proposal document and CSV file.

Technologies Used:
- Python
- Streamlit
- python-docx
- HTML/CSS

How to Run:
1. Install dependencies
2. Run: streamlit run app.py
3. Open browser at localhost:8501
